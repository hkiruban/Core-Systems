# Database Logic Foundations

### The SQL Query Anatomy
| Component | Role | Analogy |
| :--- | :--- | :--- |
| **SELECT** | The "What" | Picking which items to put in your shopping cart. |
| **FROM** | The "Where" | Choosing which aisle (table) you're shopping in. |
| **WHERE** | The "Filter" | Only picking items that are on sale (specific conditions). |

### Objectives
* Master set-based logic
* Optimize filtering to handle data.
