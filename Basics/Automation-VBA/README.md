# VBA & Automation Foundations

### The VBA Data Anatomy
| Component | Role | Analogy |
| :--- | :--- | :--- |
| **Sub / End Sub** | **The Procedure** | Defining the boundaries of a specific data task. |
| **Dim** | **Memory Allocation** | Reserving space in the system for your data types. |
| **Range / Cells** | **Coordinate Mapping** | Pinpointing the exact address of the data on the grid. |
| **If / Then** | **Conditional Logic** | Filtering outliers or identifying specific market signals. |
| **For / Next** | **Iterative Processing** | Automating the same calculation across thousands of rows. |

### Objectives
* Automate manual Excel workflows to reduce human error.
* Implement data cleaning and normalization for datasets.
